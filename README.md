# spacecraft-docking
A futuristic 2D spacecraft docking simulator built using HTML, CSS, and JavaScript.

Control a spacecraft using realistic inertia-based movement and safely dock with the station without crashing.

✨ Features
🚀 Spacecraft controls and rotation
🛰 Docking mechanics
⛽ Fuel system
📊 Live telemetry HUD
🔄 Restart mission button
🌌 Futuristic space UI
🎮 Controls
W → Thrust
A/D → Rotate
S → Slow down
🛠 Built With
HTML5 Canvas
CSS3
JavaScript
